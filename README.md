# E-commerce Sales & Revenue Prediction Dataset (SQL)

## Overview
This project contains SQL queries for analyzing an e-commerce dataset with features like:
- Region, Channel, Product Category, Customer Segment
- Ad Spend, Price, Discount Rate
- Market Reach, Impressions, CTR
- Competition Index, Seasonality Index
- Customer Lifetime Value

## Queries Included
- Revenue by Region
- Revenue by Region + Channel
- Top 5 Channels by Revenue
- Monthly Revenue Trends
- ROI Analysis (Revenue vs Ad Spend)
- Product Category Performance
- Customer Segment Profitability
- Seasonality Impact
